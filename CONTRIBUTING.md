# Contributing to InterviewGym AI

Thank you for your interest in contributing to InterviewGym AI! We welcome contributions from the community.

## How to Contribute

1.  **Fork the repository** on GitHub.
2.  **Clone your fork** locally:
    ```bash
    git clone https://github.com/tomduyanh/interview-gym-ai.git
    ```
3.  **Create a new branch** for your feature or bugfix:
    ```bash
    git checkout -b feature/your-feature-name
    ```
4.  **Make your changes** and ensure they adhere to the project's coding standards.
5.  **Commit your changes** with a descriptive commit message.
6.  **Push your branch** to your fork:
    ```bash
    git push origin feature/your-feature-name
    ```
7.  **Submit a Pull Request** to the `master` branch of the original repository.

## Reporting Bugs

If you find a bug, please create a GitHub issue with:
- A clear description of the bug.
- Steps to reproduce it.
- Expected vs. actual behavior.

## Feature Requests

Feel free to open an issue to suggest new features or improvements.

Happy coding!
